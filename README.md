# 施美玉名香祖舖網站（SHI MEI YU）

一個使用 Next.js 建立的品牌展示網站，包含商品展示、分類搜尋與品牌介紹。

---

## 技術架構

- Next.js 14 (App Router)
- React
- TypeScript
- Tailwind CSS

---

## 功能

- 商品列表（分類 / 搜尋）
- 商品詳細頁（動態路由）
- 品牌介紹頁
- 聯絡頁面
- Navbar（手機版選單）
- Scroll 動畫
- Hero 視差效果
- SEO（metadata / sitemap / robots）
